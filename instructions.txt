For Windows users need to Install Git terminal to run this program on windows

download this terminal from 
- https://git-scm.com/downloads



1)  Open Terminal and Cd to the dir which you downloaded from git hub.

2) Type Commend "NPM install" to install all the dependencies.

3) Type Commend {node spoiler.js "movie_name" number_of_seconds_to_wait_before_spoiler} to run this program

4) Example Command: node spoiler.js "Arrival" 15
